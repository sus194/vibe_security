================================================================================
                      DETECTION CAPABILITIES
================================================================================

8 SECURITY DETECTORS:

1. Secrets Detection (CRITICAL)
   - API keys (OpenAI, Stripe, Firebase, etc)
   - Cloud credentials (AWS, GCP)
   - GitHub tokens, private keys
   - .env files, hardcoded fallbacks

2. Input Validation (HIGH/CRITICAL)
   - SQL injection patterns
   - Command injection risks
   - Path traversal vulnerabilities
   - Missing schema validation

3. Authentication (HIGH/CRITICAL)
   - Unverified JWT tokens
   - Missing auth middleware
   - Lenient auth checks
   - Client-side auth trust

4. CORS (MEDIUM/CRITICAL)
   - Wildcard origins (*)
   - Insecure credential handling
   - Default permissive configs

5. Dangerous Defaults (MEDIUM)
   - Debug mode enabled
   - Hardcoded ports
   - Admin flags set to true
   - Trust proxy without validation

6. File Uploads (HIGH)
   - Missing size limits
   - No MIME type validation
   - Path traversal risks

7. Dependencies (MEDIUM)
   - npm install --force
   - Abandoned packages
   - Known vulnerabilities

8. AI Security (MEDIUM/HIGH)
   - Prompt injection risks
   - Raw user input in prompts
   - Missing output filtering
